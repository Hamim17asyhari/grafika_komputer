# Mini Project Transformasi 3D

Project ini mengimplementasikan 4 transformasi 3D dasar menggunakan Python dan library Tkinter.

## Fitur
Sesuai permintaan, project ini mencakup:

1.  **Translasi 3D**: Menggeser objek ke kiri, kanan, atas, bawah (sumbu X dan Y layar).
2.  **Rotasi 3D**: Memutar objek (Kubus) terhadap sumbu X, Y, dan Z.
3.  **Skala 3D**: Memperbesar dan memperkecil ukuran objek.
4.  **Refleksi 3D**: Mencerminkan objek terhadap sumbu X (membalik koordinat X).

## Cara Menjalankan

Pastikan Anda memiliki Python terinstal.
Jalankan perintah berikut di terminal:

```bash
python main.py
```

## Kontrol
Gunakan panel tombol di bagian bawah jendela untuk melakukan transformasi.
- **Translasi**: Tombol panah (X dan Y).
- **Rotasi**: Tombol `Rot X`, `Rot Y`, `Rot Z`.
- **Skala**: Tombol `Perbesar` dan `Perkecil`.
- **Refleksi**: Tombol `Refleksi X`.
- **Reset**: Mengembalikan objek ke posisi semula.
